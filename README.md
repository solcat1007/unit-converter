# 单位换算器 (Unit Converter)

功能全面的网页单位换算工具，支持长度、重量、温度、面积、速度和数据存储六大类别。

## 功能

- **六大类别**：长度、重量、温度、面积、速度、数据存储
- **双向换算**：自由选择输入和输出单位，支持一键交换
- **实时计算**：输入数值即时转换
- **预设常用值**：每个分类提供常用数值快速填入
- **公式展示**：显示换算公式，清晰展示换算关系
- **中英文切换**：界面和单位名称同步切换
- **支持单位**：
  - 长度：米/千米/厘米/毫米/英寸/英尺/英里
  - 重量：千克/克/毫克/吨/磅/盎司
  - 温度：摄氏度/华氏度/开尔文
  - 面积：平方米/平方千米/公顷/英亩/平方英尺
  - 速度：米/秒/千米时/英里时/节
  - 数据：字节/KB/MB/GB/TB

## 使用方法

1. 用浏览器打开 `index.html`
2. 点击顶部标签切换换算类别
3. 输入数值，选择输入和输出单位
4. 结果实时显示，包含换算公式
5. 点击 ⇄ 按钮交换输入输出单位
6. 点击预设值快速填入常用数值

## 技术栈

纯 HTML/CSS/JavaScript。
